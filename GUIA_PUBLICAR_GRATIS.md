# Guia rápido para publicar grátis

## 1. Supabase

Acesse Supabase, crie um projeto gratuito e execute o arquivo:

```txt
supabase/schema.sql
```

Depois pegue:

- Project URL
- anon public key

## 2. Vercel

Suba o projeto no GitHub e importe na Vercel.

Adicione as variáveis de ambiente:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

## 3. Celular

Depois do deploy, abra o link da Vercel no celular.

Exemplo:

```txt
https://seu-projeto.vercel.app
```

## 4. Criar conta

Na tela de login, clique em "Criar conta".

Use e-mail e senha.

Se o Supabase pedir confirmação de e-mail, confirme antes de entrar.
